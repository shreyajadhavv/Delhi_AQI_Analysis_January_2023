# Delhi_AQI_Analysis_January_2023
An in-depth exploratory and statistical analysis of Delhi's hourly air quality data, examining pollutant behavior, AQI trends, diurnal patterns, and health risk distribution using Python.
# Research Questions Addressed
1. What is Delhi's overall AQI profile during January 2023?
2. Which hour of the day records the worst air quality?
3. Is there a statistically significant difference between
   morning and evening pollution levels?
4. Which pollutants are most strongly correlated with AQI?
5. Does PM2.5 vary significantly across calendar days?
6. What are the top 10 most severely polluted hours?
7. How are pollutant distributions shaped (skewness, kurtosis)?
## 📊 Key Findings
- 98.9% of hours recorded AQI above 200 (Poor or worse)
- Peak AQI occurs at 15:00 hrs (avg = 449.16)
- PM2.5 shows the strongest correlation with AQI (r = 0.79)
- Morning and evening AQI differ significantly (p < 0.001)
- O3 (ozone) behaves inversely to primary pollutants
## 🛠️ Tools & Libraries Used
- Jupyter
- Pandas, NumPy — data manipulation
- Matplotlib, Seaborn — visualizations
- SciPy — statistical tests (t-test, ANOVA, Pearson r)
- Scikit-learn — data normalization
- Power BI — interactive dashboard
## 📁 Dataset
- Period: January 1–24, 2023 (hourly, 561 records)
- Columns: date, co, no, no2, o3, so2, pm2_5, pm10, nh3
